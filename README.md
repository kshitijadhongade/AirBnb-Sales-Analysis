# AirBnb-Sales-Analysis
This project analyzes Airbnb listings, availability, and reviews data to uncover insights into pricing strategies, guest satisfaction, and host performance. Using Tableau, I built interactive dashboards that allow recruiters, analysts, and business stakeholders to explore trends in property types, review sentiment, and availability patterns.
├── data/
│   ├── listings.csv
│   ├── availability.csv
│   └── reviews.csv
├── tableau/
│   └── Airbnb_Analysis.twbx
├── README.md

📈Dashboards
- Pricing & Availability Trends
- Daily price fluctuations
- Availability across 30/60/90/365-day windows
- Guest Reviews & Sentiment
- Review volume over time
- Word cloud of guest comments
- Average ratings by property type
- Host Performance
- Response rate vs. review scores
- Superhost analysis
- Listings count per host

🚀 How to Use
- Download the Tableau workbook (Airbnb_Analysis.twbx)
- Open in Tableau Desktop or Tableau Public
- Explore dashboards interactively:
- Filter by city, neighborhood, or property type
- Drill down into individual listings
- Compare pricing vs. review scores

🛠️ Tools & Skills
- Tableau: Dashboard design, interactivity, storytelling
- SQL & Excel: Data cleaning and preprocessing
- Python (optional): Sentiment analysis on reviews
- GitHub: Version control and portfolio showcase

📌 Key Insights
- Listings with higher review scores often correlate with consistent pricing strategies
- Superhosts tend to maintain higher availability and guest satisfaction
- Seasonal demand significantly impacts pricing and booking windows
